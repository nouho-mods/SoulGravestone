# Soul Gravestone

This mod is designed to improve the death experience by preventing item loss and making recovery less frustrating. Like most gravestone mods this mod will create a grave on death, safely storing your items and remaining experience until you return. The mod also features a unique Soulshape effect, allowing you to temporarily return as a ghost on respawn.

## Features
- Places a gravestone block at your death location, storing your items and XP
- Soulshape effect: temporarily return as a ghost after death to help recover your grave
- Gives a recovery compass on respawn
- Edge case handling so that the gravestone is always retrievable
- Configurable behavior for deaths in lava, water, or the void
- Configurable Duration and abilities of the Soul Shape effect
- Configurable amount of XP stored in gravestone

## Compatibility
- Required on both server and client
- Built in Curios compatibility
- Avoid using this with other Gravestone mods installed
