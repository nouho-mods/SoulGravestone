# Changelog

All notable changes to this project will be documented in this file.


## [1.0.1] - 2025-10-30
### Fixed
- Death by explosion destroying gravestones.
- Getting SoulShape effect when going through the end exit portal.

## [1.0.2] - 2025-11-01
### Fixed
- Gravestone inventory not being saved when closing the game/server.
- Added fallbacks when gravestone creation fails (vanilla behavior).
- Added retry for gravestone item storage in case inventory is not initialized yet.
- Improved death by explosion logic. 
- Added more debug logs for future issues.

## [1.0.3] - 20XX-XX-XX
### Fixed
- Enhanced compatibility with some mods that prevent your death in quirky ways. (See Ars Additions's Second Wind)
